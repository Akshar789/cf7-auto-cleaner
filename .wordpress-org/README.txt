# WordPress.org Assets

Place the following files in this directory for WordPress.org plugin page:

## Required Images:

1. **icon-128x128.png** - Plugin icon (128x128 pixels)
2. **icon-256x256.png** - Plugin icon retina (256x256 pixels)
3. **banner-772x250.png** - Plugin banner (772x250 pixels)
4. **banner-1544x500.png** - Plugin banner retina (1544x500 pixels)

## Optional Screenshots:

Place screenshots as:
- screenshot-1.png (Settings page)
- screenshot-2.png (Per-form settings)
- screenshot-3.png (Client-side filtering in action)
- screenshot-4.png (User notification)

These will be automatically displayed on the WordPress.org plugin page.
